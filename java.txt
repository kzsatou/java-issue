1. Java と C++ の違いを調べてみましょう
・C++
ポインタがあり、メモリを直接扱うことができる
IoT向け
ハードウェア、コンピュータの知識が必要
・Java
メモリを直接扱うことはできない
Webシステム、アプリ開発向け
プラットフォームに依存しない

2. Java と JavaScript の違いを調べてみましょう
・JavaScript
動きのあるWebページを開発する際に使われる
コンパイルが不要
フロントエンド分野で使われる
データ型を適宜変更できる動的型
・Java
業務システム、Webサービスの開発に利用される
コンパイルが必要
データ型を最初に固定で決める静的型

3. Java と Python の違いを調べてみましょう
・Python
コードの記述がシンプルで分かりやすい
ソースコードが少ない
数値計算や統計処理に関するライブラリが豊富
実行速度が他の言語と比較して遅い
・Java
大規模開発に向いている(汎用性が高い)
